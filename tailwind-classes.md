# Tailwind классы — краткая сводка по проекту

---

## Позиционирование

| Класс | Что делает |
|---|---|
| `relative` | Родитель для absolute-элементов. Absolute дети считают координаты от него |
| `absolute` | Вырывает элемент из потока, позиционируется относительно ближайшего `relative` родителя |
| `z-10` | Слой по оси Z — элемент рисуется поверх тех, у кого z меньше |
| `top-[40px]` | Отступ сверху (произвольное значение в скобках) |
| `left-0` | Прижать к левому краю родителя |

---

## Размеры и сетка

| Класс | Что делает |
|---|---|
| `w-full` | Ширина 100% от родителя |
| `h-full` | Высота 100% от родителя |
| `min-h-screen` | Минимальная высота = высота экрана |
| `grid` | Включает CSS Grid |
| `grid-cols-2` | Два равных столбца |
| `grid-cols-1` | Один столбец (используется на мобиле) |

---

## Отступы

| Класс | Что делает |
|---|---|
| `p-8` | Padding со всех сторон 32px |
| `px-12` | Padding по горизонтали (left + right) 48px |
| `py-3` | Padding по вертикали (top + bottom) 12px |
| `mt-5` | Margin top 20px |
| `mb-12` | Margin bottom 48px |
| `space-y-8` | Вертикальный gap между дочерними элементами 32px |
| `pt-[220px]` | Padding top произвольный — нужен чтобы контент не залезал под absolute заголовок |

---

## Типографика

| Класс | Что делает |
|---|---|
| `font-bold` | font-weight: 700 |
| `font-medium` | font-weight: 500 |
| `uppercase` | Все буквы заглавные |
| `whitespace-nowrap` | Запрещает перенос строки — текст в одну строку |
| `tracking-[-0.015em]` | Межбуквенный интервал (отрицательный = буквы ближе) |
| `leading-[1]` | Межстрочный интервал = 1 (плотно) |
| `text-[200px]` | Размер шрифта произвольный |
| `text-xs` | Размер шрифта 12px |
| `text-sm` | Размер шрифта 14px |
| `text-left` | Выравнивание текста по левому краю |
| `text-right` | Выравнивание текста по правому краю |
| `text-center` | Выравнивание по центру |
| `font-['Inter',_sans-serif]` | Задаёт конкретный шрифт |

---

## Цвет и фон

| Класс | Что делает |
|---|---|
| `text-black/80` | Чёрный цвет текста с 80% непрозрачностью |
| `text-white` | Белый текст |
| `text-gray-400` | Серый текст (используется для placeholder) |
| `bg-[#600a0a]` | Фон произвольным HEX-цветом (тёмно-бордовый) |
| `bg-white` | Белый фон |

---

## Флекс

| Класс | Что делает |
|---|---|
| `flex` | Включает Flexbox |
| `flex-col` | Дочерние элементы в колонку (вертикально) |
| `items-center` | Выравнивание по поперечной оси (по центру) |
| `justify-center` | Выравнивание по главной оси (по центру) |
| `justify-end` | Прижать содержимое к концу главной оси |

---

## Границы

| Класс | Что делает |
|---|---|
| `border-b` | Граница только снизу |
| `border-black` | Цвет границы — чёрный |
| `outline-none` | Убирает стандартный браузерный outline у input при фокусе |

---

## Изображения

| Класс | Что делает |
|---|---|
| `object-cover` | Картинка заполняет контейнер, сохраняя пропорции (обрезает лишнее) |
| `block` | display: block — убирает лишний отступ снизу у img |

---

## Интерактивность

| Класс | Что делает |
|---|---|
| `hover:bg-black` | Фон становится чёрным при наведении |
| `transition-colors` | Плавная анимация смены цвета |

---

## Адаптив (брейкпоинты)

| Префикс | Ширина экрана |
|---|---|
| `sm:` | от 640px |
| `md:` | от 768px |
| `lg:` | от 1024px |
| `xl:` | от 1280px |

Пример: `md:grid-cols-2` — два столбца только начиная с 768px, до этого один.

---

## Placeholder

| Класс | Что делает |
|---|---|
| `placeholder:text-gray-400` | Цвет текста placeholder серый |
| `placeholder:font-medium` | Жирность placeholder font-weight: 500 |
